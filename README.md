# 自动化连点器

一个基于 Java 17 和 Swing 的基础桌面连点器。

## 功能

- 简洁的自适应桌面界面与菜单栏
- 全局快捷键：F3 保存坐标、F4 启动、F5 暂停/继续、F6 立即停止
- 双击任一坐标点，编辑它独立的开始延迟、连续点击间隔和点击次数
- “设置 → 设置任务界面”可设定新点位默认参数、任务循环次数，并可批量应用到已有点位
- 支持删除单个点位或清空全部点位
- 配置自动保存；支持导入、导出、另存为和任务执行报告导出
- 帮助菜单包含使用说明、打开日志文件和检查更新入口
- 显示本次点击数量
- 自动在当前鼠标位置执行左键单击

## 在 IDEA 中运行

1. 选择 **File → Open**，打开本项目的 `pom.xml`。
2. 使用 Java 17 SDK 导入 Maven 项目。
3. 运行 `AutoClickerApp` 的 `main` 方法。

也可以在项目根目录执行：

```shell
mvn compile exec:java
```

将鼠标移到目标位置后按 **F3**，即可记录点位；双击列表项可修改该点位参数。打开 **设置 → 设置任务界面** 可设置之后创建点位的默认值、任务点位循环次数（`0` 表示无限循环），也可选择将默认值批量应用到已有点位。配置会自动保存在用户目录的 `.clickflow/config.properties`，下次启动自动恢复；也可以通过“文件”菜单导入、导出或另存为。本版本仅供个人桌面自动化或测试使用。
